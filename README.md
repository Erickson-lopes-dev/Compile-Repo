# Compile-Repo

## instalação

From source:

```
git clone https://github.com/Erickson-lopes-dev/Compile-Repo
cd Compile-Repo
pip install -r requeriments.txt

```
## Exemplo

Insira dentro do arquivo 'repositories.txt' os repositórios que deseja analisar

em seguida execute o arquivo main:
'''
py main.py
'''
